Install
-------

Linux
-----		
	1. Ubuntu(debian like)::

		sudo apt-get install pandoc

	2. Fedora/Red Hat/CentOS::

		yum install pandoc

	3. Archlinux::

		pacman -S pandoc

	4. Or download pandoc here: https://github.com/jgm/pandoc/releases

Mac or Windows
--------------

	1. Download pandoc here: https://github.com/jgm/pandoc/releases
